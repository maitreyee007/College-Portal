# College-Portal
Responsive College Management System landing page built with HTML, CSS, and JavaScript.
Build a modern, responsive College Management System Landing Page using only HTML5, CSS3, and Vanilla JavaScript. Create the entire project from scratch in an empty folder without using Bootstrap, Tailwind CSS, React, Angular, Vue, jQuery, or any external UI frameworks.

The website should be inspired by the professional look and feel of modern educational institution websites (such as PSVPEC) but must be an original design with unique layout, styling, and code.
